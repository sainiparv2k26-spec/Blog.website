About

A simple static newsletter website for Calgary-focused writing.

YYC Daily is built as a public, read-only newsletter site. New issues are added manually by editing the code, making it lightweight, simple, and easy to run on GitHub Pages.

It is ideal for publishing daily or weekly newsletters without needing any backend, login system, or admin panel.

How posting works

To publish a new newsletter, edit the posts array inside script.js, save, commit, and push to GitHub.
